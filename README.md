# AI Resume Screening System (ATS)

An AI-powered Applicant Tracking System built with Django that automates resume screening, scoring, and candidate ranking.

## 🚀 Features
- Resume PDF Upload
- AI-based Skill Extraction
- Job Matching System
- Candidate Scoring Engine
- Talent Pool Management
- Ranking Dashboard
- Candidate Detail Analysis

## 🧠 Tech Stack
- Python (Django)
- NLP (Sentence Transformers)
- HTML / CSS (SaaS UI)
- SQLite

## 📊 System Flow
Upload Resume → Extract Text → AI Scoring → Job Matching → Talent Pool → Ranking Dashboard
